🌐 What is Blockchain?

A Blockchain is a digital ledger made of blocks that store information. It is:

  • Decentralized – no single person or authority controls it
  • Immutable – once data is recorded, it can't be changed easily
  • Transparent – all participants can verify data

Think of it like a shared Google Doc — everyone sees the same copy and every change is logged.

🧱 What is a Block?

A Block is a container for data. In a blockchain, each block typically stores:

  • Index: Position of the block in the chain
  • Timestamp: When it was added
  • Data: Such as transaction details
  • Previous Hash: The hash of the previous block (acts like a digital fingerprint)
  • Hash: A cryptographic summary of this block’s contents

Each block is connected to the previous one through its hash, forming a chain.

🔐 What is a Hash?

A Hash is a special function that takes any data and produces a fixed-length code (usually in hexadecimal format). Example: the SHA-256 hash of the word "hello" is: 

2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

Key characteristics of a hash:

  • Deterministic – same input gives same output
  • Irreversible – you can’t get the original data back from the hash
  • Unique – a small change in input gives a completely different hash

🔗 What is a Blockchain?

It is a chain of blocks where each block is linked to the previous one using a hash. This linkage ensures:

  • Security – tampering with one block breaks the entire chain
  • Integrity – all blocks are verified through consensus
  • Traceability – you can track the entire history

🧬 Types of Blockchain

1. Public Blockchain

Open to everyone. Anyone can read/write to the blockchain and participate in the consensus mechanism.

  • Examples: Bitcoin, Ethereum
  • Highly secure but slower and resource-intensive

2. Private Blockchain

Controlled by a single organization. Participation is restricted.

  • Examples: Used by banks, supply chains
  • Faster and more scalable, but less decentralized

3. Consortium (Federated) Blockchain

Governed by a group of organizations instead of a single entity.

  • Examples: Hyperledger, R3
  • Common in healthcare, finance, and logistics

4. Hybrid Blockchain

Combines public and private blockchains.

  • Example: IBM Food Trust
  • Offers privacy + transparency as needed
